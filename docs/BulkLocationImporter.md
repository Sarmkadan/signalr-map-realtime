# BulkLocationImporter

Utility for bulk-importing location data from CSV files into a SignalR real-time map application. Handles parsing, validation, and asynchronous persistence of location records while providing sample data generation for testing.

## API

### `BulkLocationImporter`
Public parameterless constructor. Initializes a new instance of the importer with default configuration for CSV processing and location validation.
